# Curso-IoT-BluetoothECG

Bluetooth ECG - Mostrando seu batimendo cardíaco com JavaScript e Arduíno

Professor: [Daniel Furini](https://github.com/furini)

## Objetivo

Projeto tem como objetivo capturar a frequência luminosa da corrente sanguínea e enviar os dados via Bluetooth para um Android.

## Requisitos

­ Arduino ou Compatível
­ Pulse Sensor, módulo Bluetooth ( HC­05 ou HC­06 ) ­ Android 2.3+
­ Cordova ( instalado e configurado para a plataforma Android )
­ Plugin de Bluetooth
­ Plugin de Crosswalk
Configurações:
­ Configurar módulo de Bluetooth ( Comando AT )

## Ementa

­ Ligação/Pinagem Arduino
­ Criando o código Arduino.
­ Criando o código HTML5 ( CSS, Canvaas, JS ) para interpretar os dados.
